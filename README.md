# Snake (Windows Forms)

Небольшой учебный проект «Змейка» на **C# / .NET 6** с редактором карт.
Решение состоит из нескольких проектов: запуск игры, конструктор карт и общая игровая логика.

## Технологии

- **C# 10**
- **.NET 6 (`net6.0-windows`)**
- **Windows Forms** для интерфейса
- **System.Drawing** для отрисовки поля, змейки и еды
- **Сериализация в файл (`Maps.dat`)** для сохранения пользовательских карт

## Структура решения

- `MapLoader` — стартовое приложение, показывает список карт, позволяет:
  - создать новую карту,
  - отредактировать существующую,
  - удалить карту,
  - запустить игру на выбранной карте.
- `MapConstructor` — визуальный редактор карты:
  - настройка размера сетки,
  - рисование препятствий,
  - задание начального тела/направления змейки.
- `Game` — игровой цикл и отрисовка текущей партии.
- `Core` — модели и правила игры (`Map`, `Snake`, движение, столкновения, еда).

## Как это работает

1. При запуске `MapLoader` считывает карты из `Maps.dat` (если файл есть).
2. Пользователь выбирает карту или создаёт новую через редактор.
3. В `MapConstructor` сохраняются:
   - имя карты,
   - размер поля,
   - занятые клетки (препятствия),
   - стартовое состояние змейки.
4. При старте игры создаётся копия змейки, спавнится еда в свободной клетке, затем запускается цикл:
   - змейка двигается по направлению,
   - при выходе за границы поля появляется с противоположной стороны,
   - при столкновении с собой/препятствием игра заканчивается,
   - при поедании еды змейка растёт и появляется новая еда.
5. При закрытии `MapLoader` список карт сериализуется обратно в `Maps.dat`.
